# dotfiles
just copy paste the vs code jsons and follow the instruction below to set up zsh

### zsh config
If this repository resides in the /Users/Shared/ directory on macos and you are in your user directory, execute:
1. echo "source /Users/Shared/dotfiles/.zshrc" > .zshrc
2. ln -s /Users/Shared/dotfiles/.aliases .aliases
3. install required plugins
