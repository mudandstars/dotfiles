### zsh config
If this repository resides in the /Users/Shared/ directory and you are in your user directory, execute:
```bash
1. echo "source /Users/Shared/dotfiles/.zshrc" > .zshrc
2. ln -s /Users/Shared/dotfiles/.aliases .aliases
3. install required plugins
```
